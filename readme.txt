== Installation ==

1. Upload the `woocommerce-custom-payment` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the WC-custom-payment for WP.
4. Set you Secret key and click save changes. Select the forms from the dropdown that you'd like.
5. Once done, please visit forms section and select the form needed.
6. You can select forms with embed code or shotcode. Place it into any page and publish the page.
